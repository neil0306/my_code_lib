# 数组的经典考察目标
数组在C++中是一段连续的内存块, 它的元素不能删除, 只能覆盖.

对于数组, 一般考察4中经典问题:
```txt
1. 查找
    一般使用二分法.
        - 时间复杂度 O(log n)

2. 移除元素
    一般使用双指针.
        - 快慢指针
        - 左右(前后)指针
        - 时间复杂度 O(n)

3. 数组中满足某些条件的元素集合
    滑动窗口
        - 也算是双指针的一种, 使用滑动窗口需要留意3个问题:
            - 窗口里的元素需要满足什么条件?
            - 窗口的左边界怎么移动? 
            - 窗口的右边界怎么移动? (是随着遍历数组的指针移动还是怎么搞?)
        - 时间复杂度 O(n)

4. 模拟某种过程
    - 螺旋矩阵的 生成 and 打印元素
    - 需要遵循`循环不变量`的原则: 
        - 遍历的过程中一定不能改变的规则是什么?
            - 螺旋矩阵的操作中, 这个规则一般是遍历的区间, 选择 "左闭右开" 还是 "左闭右闭".
```
